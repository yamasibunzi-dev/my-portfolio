2. Snakes and Ladders
View

I created a digital version of Snakes and Ladders using Java in Apache NetBeans IDE 17 to explore object-oriented programming and interactive game design. This project features a graphical board, dice roll simulation, player movement logic, and dynamic interactions with snakes and ladders, all built using Java Swing. It helped me strengthen my understanding of arrays, conditionals, and event handling while applying creative problem-solving to build a fun, functional game. The project reflects my multidisciplinary approach at CPUT, blending logic, design, and user experience into a playful learning tool that showcases my growth as a developer.

The game reflects my creative and goal-driven approach at CPUT, blending technical skills with playful design to challenge users and showcase my growing expertise in Java development.
